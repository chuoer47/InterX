# 蓝牙激光鼠标 使用说明书
![图示](插图/Manual27_0.jpg)

## 关于本手册
感谢您购买本款蓝牙鼠标套装。本手册涵盖的主题如下。（本款鼠标销售时不含 USB 蓝牙适配器。）
- 产品介绍
- 电池安装
- WIDCOMM 蓝牙协议栈的安装与卸载
- 激光鼠标设置
- 激光鼠标使用
- WIDCOMM 蓝牙协议栈使用
- 故障排除

## 1. 产品介绍
### 产品特性
激光鼠标特性：
- 带滚轮、左右键及侧倾滚轮
- 省电模式
- LED 指示灯
  - 琥珀色：电量低
  - 绿色闪烁：鼠标正在配对

### USB 蓝牙适配器特性
- 兼容蓝牙 v1.2 及 v2.0 规范
- 支持鼠标设备
- 支持其他蓝牙设备

### 系统要求
- 支持蓝牙 / USB 鼠标的电脑，运行 Microsoft Windows 98/ME/2000/XP 或更高版本

## 2. 安装电池
鼠标需使用两节 AA 型号电池。请按照以下步骤安装电池。
1. 按下按钮，电池仓盖弹出。
![图示](插图/Manual27_1.jpg)
2. 按照鼠标内部标注的正负极（+ 和 -）装入两节 AA 电池。
![图示](插图/Manual27_2.jpg)
3. 装回电池仓盖。
![图示](插图/Manual27_3.jpg)

## 3. WIDCOMM 蓝牙协议栈的安装与卸载
### 3.1 安装 WIDCOMM 蓝牙协议栈
安装 WIDCOMM 蓝牙协议栈前，请将 USB 蓝牙适配器插入电脑的 USB 接口。
1. 将附带的安装光盘放入光驱。
2. 安装程序将自动加载。若未自动加载，请双击光盘 `bin` 文件夹内的 `Setup.exe`。
3. 欢迎界面将引导您完成安装。点击“下一步”继续。
![图示](插图/Manual27_4.jpg)
4. 阅读协议并选择“我接受许可协议中的条款”。点击“下一步”继续。
![图示](插图/Manual27_5.jpg)
5. 程序将安装至图示文件夹。点击“更改”可选择其他安装路径。点击“下一步”继续。
![图示](插图/Manual27_6.jpg)
6. 点击“安装”开始安装。安装过程可能需要一些时间。
![图示](插图/Manual27_7.jpg)
7. 出现以下界面表示安装完成。
![图示](插图/Manual27_8.jpg)
8. 拔下 USB 蓝牙适配器后重新插入，鼠标将自动配对。点击“完成”退出安装。系统将提示您重启电脑以使蓝牙协议栈生效。点击“是”立即重启，或“否”稍后重启。
![图示](插图/Manual27_9.jpg)

### 3.2 卸载 WIDCOMM 蓝牙协议栈
请按照以下步骤从电脑中卸载 WIDCOMM 蓝牙协议栈。
1. 拔下 USB 蓝牙适配器。
2. 点击 开始 > 设置 > 控制面板 > 添加或删除程序。
3. 在程序列表中选择 WIDCOMM 蓝牙软件，点击“删除”。
4. 系统开始卸载程序，此过程可能需要一些时间。
5. 程序卸载完成后，系统将提示重启电脑。点击“是”立即重启，或“否”稍后重启。

## 4. 激光鼠标设置
### 4.1 Windows 系统快速配对
蓝牙鼠标出厂默认已设置为预配对模式。如需快速配对，请按照以下步骤启用预配对模式。
1. 将 USB 蓝牙适配器插入电脑的 USB 接口。
2. 按下鼠标底部的配对按钮。
![图示](插图/Manual27_10.jpg)
3. 按下 USB 蓝牙适配器底部的按钮，直至 LED 指示灯每秒闪烁一次。这表示蓝牙适配器已开始搜索待配对设备。（本款鼠标销售时不含 USB 蓝牙适配器。）
![图示](插图/Manual27_11.jpg)
4. 移动鼠标以确认连接成功。

### 在 BIOS 中启用 USB 鼠标功能
若鼠标无法使用（更多可能情况请参阅第 14 页“7. 故障排除”），请检查 BIOS 设置中是否已启用鼠标的 USB 功能。部分老旧电脑系统在 BIOS/DOS 环境下不支持 USB 键盘 / 鼠标。

### 4.2 通过 WIDCOMM 蓝牙协议栈配对
使用蓝牙协议栈进行鼠标配对前，请先按照“3. WIDCOMM 蓝牙协议栈的安装与卸载”中的说明安装协议栈。然后按照以下步骤操作。
1. 按下 USB 蓝牙适配器按钮，直至出现人机接口设备（HID）界面。
![图示](插图/Manual27_12.jpg)
2. 鼠标连接成功后，将出现以下确认信息。点击按钮确认已连接正确的鼠标。
![图示](插图/Manual27_13.jpg)

## 5. 激光鼠标使用
按照“4. 激光鼠标设置”中的说明插入 USB 蓝牙适配器并完成鼠标蓝牙连接环境设置后，您即可立即开始使用鼠标。
### 5.1 激光鼠标电量状态
蓝牙鼠标配备电量 LED 指示灯。当电量耗尽时，LED 指示灯变为琥珀色。请立即更换电池。（顶部 LED 指示灯与底部指示灯状态一致）
![图示](插图/Manual27_14.jpg)
如需查看更详细的鼠标电量信息，请前往 控制面板 > 鼠标 > 蓝牙。
![图示](插图/Manual27_15.jpg)

## 6. WIDCOMM 蓝牙协议栈使用
### 6.1 首次使用协议栈
要开始使用蓝牙协议栈，请双击系统托盘中的蓝牙图标。
![图示](插图/Manual27_16.jpg)
1. 蓝牙初始配置向导将启动。请按照屏幕上的说明完成配置。
2. 蓝牙配置完成后，将出现以下 HID 界面并开始搜索蓝牙设备。
![图示](插图/Manual27_17.jpg)
3. 点击您要连接的 HID 设备。然后按照屏幕上的说明启用所选设备。

### 关于蓝牙图标
蓝牙图标用于显示 USB 蓝牙适配器状态。
1. 红色图标：未插入适配器。
2. 白色图标：适配器已插入。
3. 绿色图标：适配器已检测到 HID 设备。

### 6.2 连接其他 HID 设备
除鼠标外，蓝牙协议栈还允许您连接其他蓝牙设备。
1. 确保 USB 蓝牙适配器已插入电脑。
2. 将您要连接到蓝牙协议栈的设备（如打印机、调制解调器、掌上电脑）设置为可被发现。
3. 按下 USB 蓝牙适配器按钮，直至出现以下 HID 界面。
4. 从下拉菜单中选择要搜索的设备类型。
![图示](插图/Manual27_18.jpg)
5. 将显示检测到的设备。点击设备，协议栈将引导您启用连接。

## 7. 故障排除
若您的蓝牙鼠标出现任何问题，请检查以下事项：
- 检查系统要求，确保您的系统兼容。
- 检查鼠标电池是否安装正确。确保电池正负极与鼠标电池仓内的标识一致。
- 当鼠标 LED 指示灯亮起红色时，表示电量低。请立即更换新电池。
- 电脑安装蓝牙协议栈后（请参阅第 4 页“安装 WIDCOMM 蓝牙协议栈”），您可通过 控制面板 > 鼠标 > 蓝牙 查看鼠标电量状态。
- 若鼠标无法工作，请检查系统 BIOS 中的 USB 鼠标功能。需启用该功能鼠标才能正常使用。
- 若电脑 BIOS 设置中有鼠标相关选项，请确保设置为默认值。若您不熟悉 BIOS 操作，请参阅电脑说明书获取更多信息。
- 请在干净、平整、不光滑的表面使用鼠标，以确保光标移动流畅精准。若使用鼠标垫，请选择浅色款，深色鼠标垫会增加电量消耗。

如您有其他问题需要进一步协助，请联系经销商获取技术支持。
