# Evidence Notes: 382

Question: How can you prepare the installation of Central Processing Unit (CPU) for a motherboard?

Manual used: `agentic-rag/en-manual/Motherboard.md`

Grounding evidence:

1. `Motherboard.md:131-145` gives general precautions before installing motherboard components: unplug power, avoid static electricity with a grounded wrist strap or grounded metal object, hold components by their edges, use antistatic handling when uninstalling, and ensure the ATX power supply is off or the power cord is detached.
2. `Motherboard.md:175-181` identifies the CPU section and states that the motherboard uses a surface-mount LGA1151 socket; all power cables must be unplugged before CPU installation, and only an LGA1151 CPU should be used.
3. `Motherboard.md:183-187` requires checking that the PnP cap is present and socket contacts are not bent, contacting the retailer if the cap/socket/components are damaged, keeping the cap for RMA, and notes warranty limits for incorrect installation/removal or PnP cap issues.
4. `Motherboard.md:191-203` covers CPU installation precautions: the CPU must click firmly into the CPU Installation Tool before socket installation; the tool must be used only for CPU installation, must not be bent or damaged, and should be held by both sides.
