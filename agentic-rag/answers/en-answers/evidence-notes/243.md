Question: How do I use the battery conversion feature before sailing?

Manual guess: Yamaha 210FSH 2021.md

Evidence:
- `agentic-rag/en-manual/Yamaha 210FSH 2021.md:545-568` explains the battery switch assembly in the battery compartment. It uses two marine batteries: start and house. The assembly has `"START"`, `"HOUSE"`, and `"EMERG PARALLEL"` switches. In normal operation, `"START"` and `"HOUSE"` stay ON, while `"EMERG PARALLEL"` stays OFF. The emergency parallel switch is used only if the start battery is discharged, and should be turned OFF after the engine starts or the start battery is charged.
- `agentic-rag/en-manual/Yamaha 210FSH 2021.md:557-566` identifies the switch labels and ON/OFF position colors, adjacent to `Manual09_43` and `Manual09_44`.
- `agentic-rag/en-manual/Yamaha 210FSH 2021.md:1791-1797` gives the pre-operation battery check: terminals undamaged, leads connected properly, and batteries fully charged/good enough to start the engines.
- `agentic-rag/en-manual/Yamaha 210FSH 2021.md:1923` says to turn the battery switch to ON during the engine-start sequence before operating the blowers.

Image decisions:
- `Manual09_43` used for the physical battery switch assembly labels: HOUSE, EMERG PARALLEL, START.
- `Manual09_44` used for ON/OFF position indicators: green ON and red OFF.

Answer summary:
Use normal battery switch positions before boating: START ON, HOUSE ON, EMERG PARALLEL OFF. Use EMERG PARALLEL only to start with a discharged start battery, then return it OFF. Check battery condition and connections before operating.
