Question: How can you set the onboard LED of a motherboard?

Manual guess: Motherboard.md

Evidence refs:

- agentic-rag/en-manual/Motherboard.md:663-665 The model name LEDs can be turned off or changed from BIOS or the LED Control app in Ai Suite 3; BIOS path is Advanced > Onboard Devices Configuration > Model Name LED Lighting.
- agentic-rag/en-manual/Motherboard.md:675-677 The SupremeFX LED can be turned off or changed from BIOS or the LED Control app in Ai Suite 3; BIOS path is Advanced > Onboard Devices Configuration > SupremeFX LED Lighting.
- agentic-rag/en-manual/Motherboard.md:1601 Model Name LED Lighting options are Disabled, Still Mode, and Breathing Mode.
- agentic-rag/en-manual/Motherboard.md:1615-1617 SupremeFX LED Lighting options are Disabled, Still Mode, Breathing Mode, and Flowing Mode.
