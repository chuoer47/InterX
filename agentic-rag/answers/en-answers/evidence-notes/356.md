Answer basis:
- Handset events: On = audio boost on; Off = in a call or standby; Flashing = new message, new voicemail, missed call, or muted.
- Handset charging: On = handset on base/charging cradle in standby or audio boost during a call; Off = handset off base/charging cradle; Flashing = muted.
- XL495 base station: Steady on = no new message and memory not full, handset switched on, or paging mode; Flashing = new message with memory not full, incoming call, recording, playback, or remote message access/listening from handset; Fast flashing = no new message and memory full, or registration mode.
