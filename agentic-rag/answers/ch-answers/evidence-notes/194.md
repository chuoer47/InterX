# 证据笔记 - 问题 194

## 问题
遇到温控器故障时，可采用哪些故障排除解决方案？

## 路由
产品：可编程温控器（RTH 7500型）
手册：agentic-rag/ch-manual/可编程温控器手册.md

## 检索关键词

## 证据来源

### 证据1：故障排除章节（第237-243行）
手册明确列出五种故障场景及解决方案：
1. 屏幕空白/无法切换制冷模式 → 安装全新AA碱性电池 + 检查系统设置选项220（设为1或2）
2. 需要供暖时风扇不启动 → 检查系统设置选项205
3. 制冷模式下供暖系统运行 → 检查系统设置选项200或218
4. 供暖或制冷系统无响应 → 检查系统键、温度设定、断路器、电源开关、炉门、等待5分钟
5. 供暖/制冷开启闪烁 → 压缩机保护已启动，等待5分钟 + 检查选项200或218

### 证据2：警报界面章节（第222-234行）
- 主界面显示警报图标时可在菜单查看错误/警报代码
- 暂态警报取消7天后再次显示
- 警报代码表（Manual36_49）可对照确定问题

### 证据3：更换电池章节（第192-196行）
- 电池低电量图标出现后约两个月电池将耗尽
- 未出现图标也应每年更换一次
- 两分钟内完成更换无需重新设置时间和日期
- 建议使用碱性电池

证据充分，五种常见故障场景均有明确的解决方案。电池检查和警报代码查看作为补充排查手段也有明确依据。
