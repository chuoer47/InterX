# 证据笔记 - 问题 212

问题：如何通过WIDCOMM蓝牙驱动程序为蓝牙激光鼠标设置配对？

## 路由
手册：蓝牙激光鼠标手册.md（产品别名：蓝牙激光鼠标）

## 关键证据

### 证据1：WIDCOMM蓝牙协议栈安装（第41-56行）
文件：蓝牙激光鼠标手册.md
章节：3.1 安装 WIDCOMM 蓝牙协议栈
- 安装前需将USB蓝牙适配器插入电脑USB接口
- 将安装光盘放入光驱，程序自动加载；若未自动加载，双击bin文件夹内的Setup.exe
- 按照安装向导完成安装（接受许可协议、选择路径、点击安装）
- 安装完成后拔下USB蓝牙适配器后重新插入，鼠标将自动配对
- 系统提示重启电脑使协议栈生效
- 相关图片：Manual27_4至Manual27_9

### 证据2：通过WIDCOMM蓝牙协议栈配对（第79-84行）
文件：蓝牙激光鼠标手册.md
章节：4.2 通过 WIDCOMM 蓝牙协议栈配对
- 使用协议栈配对前需先安装协议栈
- 步骤1：按下USB蓝牙适配器按钮，直至出现人机接口设备（HID）界面
- 步骤2：鼠标连接成功后，将出现确认信息，点击按钮确认已连接正确的鼠标
- 相关图片：Manual27_12（HID界面）、Manual27_13（确认信息）

### 证据3：首次使用协议栈（第95-101行）
文件：蓝牙激光鼠标手册.md
章节：6.1 首次使用协议栈
- 双击系统托盘中的蓝牙图标启动协议栈
- 蓝牙初始配置向导启动，按照屏幕说明完成配置
- 配置完成后出现HID界面并开始搜索蓝牙设备
- 点击要连接的HID设备，按照屏幕说明启用所选设备
- 相关图片：Manual27_16（蓝牙图标）、Manual27_17（HID搜索界面）

### 证据4：快速配对模式（第67-71行）
文件：蓝牙激光鼠标手册.md
章节：4.1 Windows 系统快速配对
- 蓝牙鼠标出厂默认已设置为预配对模式
- 可按下鼠标底部的配对按钮手动触发配对
- 相关图片：Manual27_10（配对按钮）

## 图片列表
- Manual27_16：系统托盘中的蓝牙图标
- Manual27_17：HID设备搜索界面
- Manual27_12：按下适配器按钮后出现的HID界面

## 答案逻辑
1. 前提条件：安装WIDCOMM蓝牙协议栈（证据1）
2. 配对方法一：通过协议栈首次使用向导配对（证据3）
3. 配对方法二：通过适配器按钮直接配对（证据2）
4. 手动触发：按下鼠标底部配对按钮（证据4）
