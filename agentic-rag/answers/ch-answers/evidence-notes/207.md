# 207 证据笔记

- 问题：如何安装儿童电动摩托车的前轮？
- 路由判断：关键词“儿童摩托/前轮/把手管/前轴/垫片/车轮螺母”均命中 `agentic-rag/ch-manual/儿童电动摩托车手册.md`。

## 核心证据
1. **安装步骤**
   - 来源：`agentic-rag/ch-manual/儿童电动摩托车手册.md`，第 77-80 行。
   - 内容：将前轴穿过把手管与前轮，每侧把手管内外两侧都需安装垫片；用两个螺母固定前轴和车轮，并用随附扳手拧紧至螺母到螺纹尽头。
2. **安装后检查与调整**
   - 来源：`agentic-rag/ch-manual/儿童电动摩托车手册.md`，第 81 行。
   - 内容：检查车轮与把手管间隙；间隙过大或车轮不稳时加装额外垫片；重新安装前轴后确认车轮自由转动，卡滞时减少一片垫片。
3. **工具与准备**
   - 来源：`agentic-rag/ch-manual/儿童电动摩托车手册.md`，第 61 行。
   - 内容：组装时可能需要十字螺丝刀、钳子和锤子，这些工具不随产品提供；扳手随产品提供。
4. **相关故障提示**
   - 来源：`agentic-rag/ch-manual/儿童电动摩托车手册.md`，第 200 行。
   - 内容：车轮螺母松动会导致车轮无法与齿轮啮合，需要拧紧螺母。
