# Evidence Notes for Question 180

## Question
驾驶摩托艇时，如何避免发生碰撞事故，保障自身和水上其他人员的安全？

## Routed Manual
摩托艇手册.md（摩托艇驾驶练习指南）

## Evidence 1: 避免碰撞要点
- File: 摩托艇手册.md
- Lines: 5-20
- Key text:
  - 碰撞事故造成的伤亡人数多于个人摩托艇（PWC）的任何其他事故类型。
  - 持续观察周围人员、物体及其他船只。留意影响视线或遮挡视野的环境状况。
  - 以安全速度谨慎驾驶，与人员、物体及其他船只保持安全距离。
  - 不要紧跟在其他摩托艇或船只后方。
  - 不要靠近他人并向其喷水或溅水。
  - 避开有水下障碍物或浅水区的区域。
  - 避免急转弯或其他让他人难以避让或判断你行驶方向的操作。
  - 尽早采取行动避免碰撞。记住，摩托艇及其他船只没有刹车装置。
  - 试图转向避开物体时不要松开油门手柄——转向需要油门动力。
  - 启动摩托艇前务必检查油门与转向操控是否正常。
  - 遵守适用于摩托艇的航行规则与当地法规。

## Evidence 2: 停车距离与避让练习
- File: 摩托艇手册.md
- Lines: 122-131
- Key text:
  - 练习区域较小时，降低最大行驶速度。
  - 速度越快，停车距离越长。
  - 在不同距离与速度下重复训练，熟悉停车与转向避让障碍物的方法。

## Evidence 3: 穿越尾流与浪涌
- File: 摩托艇手册.md
- Lines: 159-176
- Key text:
  - 穿越尾流或浪涌时，调整速度并选择合适角度。通常减速并斜切尾流可减轻冲击。
  - 以45度角穿越时可能不明显，但角度更小时效果非常明显，需做好转向与平衡准备。
  - 不要在恶劣水域或恶劣天气驾驶。

## Evidence 4: 最大载重
- File: 摩托艇手册.md
- Lines: 189-192
- Key text:
  - 最大载重：总重量（含所有货物）超过220千克（485磅）时禁止驾驶。
