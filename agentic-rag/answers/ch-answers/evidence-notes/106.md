# Evidence notes for id=106

Question: 空气净化器通常有哪些模式？如何设置？这些模式有什么特点？

## Routing
Use `agentic-rag/ch-manual/手册内容总览.md` -> item 17 `空气净化器手册.md` (Carrier Room Air Purifier).

## Key evidence
- `agentic-rag/ch-manual/空气净化器手册.md` L7, L9: manual lists 18-speed control and automatic mode as product features.
- L168-L170: regular operation starts with power key; upper/lower keys adjust F1-F18.
- L174-L180: automatic mode switches with Auto key and auto-adjusts based on IAQ/VOC sensor; manual speed adjustment disabled.
- L184-L190: turbo mode is set by long-pressing upper+lower keys for 3s; pressing lower key exits turbo and returns speed to F18.
- L194-L203: sleep mode starts with Sleep key; display dims with only moon icon, default speed F1, speed still adjustable F1-F18; Sleep again restores previous mode.
