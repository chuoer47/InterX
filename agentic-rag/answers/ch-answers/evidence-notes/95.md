### 证据 1：添加洗涤剂步骤（第214-225行）
**来源**：agentic-rag/ch-manual/洗碗机手册.md

**原文**：
```
### 添加洗涤剂
请参考程序表，根据所选程序确定正确的洗涤剂用量。
**请勿向洗涤剂盒中加入溶剂**，有爆炸危险！
请按照下图所示，在即将运行机器前将洗涤剂加入洗涤剂盒。
1. 向右推动卡扣打开洗涤剂盒盖（A）
![图示](插图/Manual06_4.jpg)
2. 向洗涤剂盒中加入建议量的粉末或块状洗涤剂。
洗涤剂盒内有刻度标记，帮助您使用适量的洗涤剂。
洗涤剂盒装满容量为 **40cm³**。
根据机器装载量和 / 或餐具脏污程度，将洗涤剂加至 **15cm³ 或 25cm³** 刻度线（1）。
![图示](插图/Manual06_5.jpg)
若餐具放置时间较长且残渣已干涸，可同时加满 **5cm³** 小格（5）。
```

**关键信息**：
- 操作步骤：打开盒盖 → 加入洗涤剂 → 关闭盒盖
- 用量指导：根据装载量和脏污程度，加至15cm³或25cm³刻度线
- 特殊情况：残渣干涸时可加满5cm³小格
- 容量上限：洗涤剂盒装满容量为40cm³
- 配图：Manual06_4（打开盒盖）、Manual06_5（刻度标记）

### 证据 2：洗涤剂类型要求（第208-213行）
**来源**：agentic-rag/ch-manual/洗碗机手册.md

**原文**：
```
### 洗涤剂
机器可使用粉末或块状洗涤剂。
机器中**只能使用洗碗机专用洗涤剂**。
建议不使用含氯和磷酸盐的洗涤剂，以免污染环境。
若洗涤程序结束后餐具仍潮湿或玻璃器皿上有水垢，请联系洗涤剂生产商。
```

**关键信息**：
- 洗涤剂形式：粉末或块状
- 强制要求：只能使用洗碗机专用洗涤剂
- 环保建议：避免使用含氯和磷酸盐的洗涤剂

### 证据 3：关闭盒盖和注意事项（第226-230行）
**来源**：agentic-rag/ch-manual/洗碗机手册.md

**原文**：
```
3. 轻轻按下关闭洗涤剂盒盖，听到咔嗒声表示已关紧。
在带预洗的高温程序中使用块状洗涤剂时，请确保洗涤剂盒盖紧闭。
无预洗的短程序建议使用粉末洗涤剂，因为块状洗涤剂的溶解度随温度和时间变化。
过量使用粉末洗涤剂会导致洗涤剂无法完全溶解，并在玻璃器皿上留下划痕。
```

**关键信息**：
- 关闭方式：轻轻按下，听到咔嗒声
- 程序匹配：高温程序用块状洗涤剂需确保盖紧，短程序建议用粉末
- 用量警告：过量使用粉末洗涤剂会导致残留和划痕

## 答案要点
1. 操作流程：打开盒盖 → 加入洗涤剂 → 关闭盒盖（3步）
2. 用量指导：根据装载量和脏污程度，参考刻度线（15cm³或25cm³）
3. 洗涤剂类型：只能使用洗碗机专用洗涤剂，粉末或块状均可
4. 安全警告：禁止加入溶剂（有爆炸危险）
5. 注意事项：遵循产品包装建议，避免过量使用
